AVL Tree Data Structure

Visual Studio - Console App - C#

I created an AVLTree data structure that could accept a file of words and store them as key value pairs with the word and the word length as properties. I implemented a three-tiered structure utilizing nodes. It contains a menu system to test functions such as Insert, Find, Delete and Print. Using Timestamps, I recorded the execution times for the above functions and plotted them on a graph to analyze their average-case time complexities.
